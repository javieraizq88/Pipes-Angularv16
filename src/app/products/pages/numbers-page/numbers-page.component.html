Pipes numéricos

en ts -> public totalSells: number = 2567789.5567;
en html -> (totalSells + 1235 ) | number:'1.2-2'
number:'1.2-2' -> suma 1 y muestra 2 decimales

{{ (totalSells + 1235 ) | number:'1.2-2' }}

en ts -> public totalSells: number = 2567789.5567;
en html -> totalSells | currency
currency Agrega USD antes del numero

{{ totalSells | currency }}

en ts -> public totalSells: number = 2567789.5567;
en html -> totalSells | currency:'CLP '
se le puede agregar la palabra que necesites

{{ totalSells | currency:'CLP ' }}

en ts -> public totalSells: number = 2567789.5567;
en html -> totalSells | currency:'':'symbol-narrow'
se puede eliminar la palabra o letras antes del numero y agregar el simbolo con symbol-narrow

{{ totalSells | currency:'':'symbol-narrow' }}

en ts -> public percent: number = 0.4856;
en html -> percent | percent: '1.2-2'
percent -> crea el porcentaje del numero

{{ percent | percent: '1.2-2' }}
